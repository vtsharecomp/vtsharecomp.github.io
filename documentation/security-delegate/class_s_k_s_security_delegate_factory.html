Creates the security delegate object by a storage device pointer and a bool optional free storage device.

If freeStorageDevice sets true then storageDevice will be free up after this created object deleted.

Parameters
storageDeviceA storage device object retrieved from storage api (devices scanned).
freeStorageDeviceA bool variable to indicate if allow free up the storageDevice or not after this object deleted (default is true).
