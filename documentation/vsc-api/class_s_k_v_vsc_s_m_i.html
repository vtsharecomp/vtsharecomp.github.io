Detailed Description

DESCRIPTION

An SKVVscSMI object is responsible to support following features:

  • Reads NAND info.
  • Reads flash id.
  • Reads flash page with specific SKVFlashPageAddress and optionals (readstrong, readspare, and scrambler).
  • Reads block status with specific SKVFlashBlockAddress.
