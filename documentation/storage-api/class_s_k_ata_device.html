Detailed Description

DESCRIPTION

An object is responsible to issue ATA commands and extended ATA commands.

  • ATA commands: identify, read sector, write sector, ...
  • Extended ATA commands: read sector extend, write sector extend, ...
  • ATA security commands: unlock, freeze lock, sanitize block erase, sanitize status extend, ...

Issues sanitize block erase extended.

The command starts a block erase sanitize operation that uses the block erase method on the user data areas,
including user data areas that are not currently allocated to cause the user data to become unretrievable.

If the password matches the password previously saved by the device, then the device shall disable the user password,
and return the device to Security Disabled/Not Locked/Not Frozen state.
