Issues submit a sanitize operation or recover from a previous failed sanitize operation.
The sanitize operation types that may be supported are Block Erases, Cryto Erase, and Overwrite.
All sanitize operations are processed in the background.

Parameters
actionA SKNvmeSanitizeAction object to indicate sanitize action.
auseA SKNvmeSanitizeAUSE object to indicate allow unrestricted sanitize exit.
owpassA SKNvmeSanitizeOWPASS object to indicate overwrite pass count.
oipbpA SKNvmeSanitizeOIPBP object to indicate overwrite invert pattern between passes.
noDeallocA SKNvmeSanitizeNoDeallocate object to indicate no deallocate after sanitize.
overwritePatternA U32 object to indicate overwrite pattern.
Returns
A SKReturnCode object to specify the return code.
